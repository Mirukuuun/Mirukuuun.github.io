本地Hexo部署

1.安装node.js与git

git下载

node.js下载

安装后在cmd当中进行验证：

git --version
node -v
npm -v

未出现问题则安装成功，否则请重新安装或检查你的环境变量。

2.markdown编辑器下载

Hexo博客文章主要采用markdown语法来书写，我们需要使用markdown文本编辑器来进行文章的书写。我采用的是 vs code+markdown 插件来进行书写(vs code实在是太香辣，主要是我用习惯了qwq)。常用的其他编辑器还有

Typora下载

MarkdownPad下载

3.安装Hexo

首先我们打开cmd，输入

npm install -g hexo-cli

安装完成后，我们切换到想要存放你的博客文件的磁盘，然后输入

hexo init blog

“blog”是你想要存放的文件夹名称，可以自己定义。安装完成后，我们进入blog目录

初始化hexo

cd blog

输入 hexo s 即可启动hexo。

启动hexo

在浏览器打开它提示的网站 localhost:4000 就能够看到我们的Hexo。

初始博客页面

到这一步，最基础的本地Hexo就安装完毕了。

4.Hexo主题更换

Hexo的默认主题是landscape，若想要更换其他的主题则需要进行下载安装。我才用的是hexo-theme-yun这个主题，它有详细的使用指南，可以根据使用指南进行安装和配置。这是该主题作者的主页：云游君的小站。

5.博客的编写

安装好Hexo之后，如何进行博客的编写呢？首先我们需要在cmd中切换到你的博客文件夹根目录下，然后输入

hexo n 博客文件名称

创建博客文件

即可生成博客文件。我们可以在 blog\source_posts\ 文件夹下找到我们新创建的博客文件 test.md，我们可以使用markdown编辑器对它进行编辑。基础的markdown语法可以参考：markdown语法。

云服务器的购买与配置

1.云服务器的购买

笔者购买的是腾讯云的学生特惠机，一年只需108！！

使用其他的如阿里云等服务器也都是ok的，可以根据自己的需要和财力进行选择。

2.云服务器的设置和连接

首先我们进入到云服务器控制台，点击重置密码，设置我们服务器的密码。

设置密码

接下来就是连接到我们的云服务器。首先要下载一个xshell。（也可以使用其他的软件）

Xshell下载

安装后，点击文件->新建，名称自行填写，主机输入我们服务器的公网ip地址，然后在左侧栏选择用户身份验证，在用户名中输入root，密码填写刚刚设置的服务器密码。

设置主机ip

填写用户名和密码

点击确定，在左侧的会话栏中双击即可连接至我们的服务器。

远程连接服务器

3.云服务器的配置

安装依赖库

yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel

安装编译工具

yum install gcc perl-ExtUtils-MakeMaker package

查看git的版本

git version

删除git

yum remove git -y

下载解压最新版本

cd /usr/local/src #下载的目录 
wget https://www.kernel.org/pub/software/scm/git/git-2.28.0.tar.gz #下载最新版 
tar -zxvf git-2.28.0.tar.gz #解压到当前文件夹

编辑并安装

cd git-2.28.0    #进入文件夹
make prefix=/usr/local/git all    #编译源码
make prefix=/usr/local/git install    #安装路径

配置git的环境变量

echo 'export PATH=$PATH:/usr/local/git/bin' >> /etc/bashrc

刷新环境变量

source /etc/bashrc

查看版本号

git --version

创建git用户并修改权限

adduser milk 
passwd milk 
chmod 740 /etc/sudoers 
vim /etc/sudoers

shift+; ，然后输入set number，来显示行号，找到第100行，点击i然后执行以下修改

root ALL=(ALL) ALL 
milk ALL=(ALL) ALL

接下来，在本地你的blog文件夹下右键，点击Git Bash Here，然后输入以下内容，创建密钥。

ssh-keygen -t rsa

在它提示你的文件夹下，找到你的密钥文件id_rsa.pub，用记事本打开，复制全部内容。然后在xshell当中，执行以下命令，粘贴你的密钥。

su fuchen
mkdir ~/.ssh
vim ~/.ssh/authorized_keys

然后在本地的git中输入

ssh -v fuchen@服务器ip

并当它提示时输入服务器密码，连接至服务器。

###4.云服务器的网站配置
首先，我们在服务器上创建一个网站目录并且设置权限：

su root
mkdir /home/hexo
chown fuchen:fuchen -R /home/hexo

然后安装Nginx

yum install -y nginx
systemctl start nginx.service    #启动服务

修改Nginx配置文件

vim /etc/nginx/nginx.conf 
38 server { 
39 listen 80 default_server; 
40 listen [::]:80 default_server; 
41 server_name fuchenchenle.cn; #域名 
42 root /home/hexo; #网站目录

重启服务器

systemctl restart nginx.service

建立git仓库

su root
cd /home/fuchen
git init --bare blog.git
chown fuchen:fuchen -R blog.git

同步网站根目录

vim blog.git/hooks/post-receive

输入以下内容

#!/bin/sh
git --work-tree=/home/hexo --git-dir=/home/fuchen/blog.git checkout -f

修改权限

chmod +x /home/fuchen/blog.git/hooks/post-receive

在本地blog根目录下打开_config.yml文件进行以下修改：

deploy:
  type: git
  repository: milk@xxx.xxx.xxx.xxx:/home/fuchen/blog.git    #用户名@服务器Ip:git仓库位置
  branch: master

在本地blog根目录下，右键打开gitbash，输入以下命令及服务器密码，将hexo部署到服务器。

hexo clean
hexo g -d

即可完成部署。
